Flex-box

Браузерная поддержка на данный момент не позволяет полноценно использовать флексы на реальных проектах. Известно большое количество багов в различных браузерах (не только IE), поэтому стоит закладывать больше времени в оценку задач, для которых планируется использовать флексы.

К использованию флексов лучше подходить с позиции прогрессивного улучшения - сначала построить основной лейаут таким образом, чтобы он работал в старых браузерах, а затем в некоторых случаях применять флексы для лучшего визуального эффекта. Тогда в старых браузерах, не поддерживающих флексы, или поддерживающих только старый синтаксис (без переноса на новую строку) верстка не сломается.

Если необходимо знать, поддерживается ли последний синтаксис в браузере, то можно использовать такую проверку:

<script>
    var d = document.documentElement.style;
    if (d.flexWrap !== '' && d.WebkitFlexWrap !== '' && d.msFlexWrap !== '') {
        // Добавить класс к html или body, загрузить стили для inline-block сетки и т.п.
    }
</script>
