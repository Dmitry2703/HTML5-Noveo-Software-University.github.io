Дмитрий Мещеряков

Рамки

Свойство border

.block {
    border: 1em solid #000;
}

Синтаксис:

border: border-width border-style border-color;
  • border-width - толщина
  • border-style - стиль
  • border-color - цвет.

Рамка с определенной стороны блока

Свойства border-top, border-bottom, border-left, border-right

.block {
    border-top: 1em solid #000;
}
.block {
    border-left: 1em solid #000;
}

Разные размеры и стили для разных границ

Свойства border-width, border-style, border-color могут иметь разное число значений.

1Все стороны одинаковые
21 значение - задает свойства для верхней/нижней границы;
2 значение - задает свойства для левой/правой границы
31 значение - задает свойства для верхней границы;
2 значение - задает свойства для левой/правой границы;
3 значение - задает свойства для нижней границы
41 значение - задает свойства для верхней границы;
2 значение - задает свойства для правой границы;
3 значение - задает свойства для нижней границы;
4 значение - задает свойства для левой границы

Примеры

.block {
    border-width: 1em 2em;
    border-style: inset;
    border-color: #ccc;
}
.block {
    border-width: 2em 1.5em 3em;
    border-style: dotted solid double;
    border-color: #eee #aaa #000;
}

Контур блока

Свойство outline

.block {
    border: 1em solid #ccc;
    outline: 1em solid #000;
}

Синтаксис:

outline: outline-width outline-style outline-color;
  • outline-width - толщина
  • outline-style - стиль
  • outline-color - цвет.
Border увеличивает размеры блока, outline - нет.
Outline применяется ко всем четырем сторонам блока.

Несколько рамок у блока

Способы создания нескольких рамок у блока:

  • Использование псевдоэлементов и свойства border.
  • Использование свойств border и outline (см. предыдущий слайд).
  • Использование свойства box-shadow (см тему Тени).
  • Использование свойства background-clip (см. тему Фоновые изображения).

Скругление углов

Свойство border-radius

.block {
    border-radius: 1em;
}

Свойство border-radius может иметь разное число значений (аналогично свойствам border-width, border-style, border-color).

Допускается писать 2 значения через слэш:

.block {
    border-radius: 5em/3em;
}

Круг на css:

.block {
    border-radius: 50%;
}

Тени

Свойство box-shadow

.block {
    box-shadow: 0.5em 0.5em 1em 0.5em #aaa;
}

Синтаксис:

box-shadow: [inset] h-shadow v-shadow [blur] [spread] [color];
  • [inset] - внутренняя тень
  • h-shadow - смещение по горизонтали ( > 0 - вправо, < 0 - влево)
  • v-shadow - смещение по вертикали ( > 0 - вниз, < 0 - вверх),
  • [blur] - размытие
  • [spread] - растяжение
  • [color] - цвет

Тени

Внутренняя тень

.block {
    box-shadow: inset 0 0 0 1em #aaa;
}

Тень только с одной стороны

.block {
    box-shadow: 0 2em 2em -1em #000;
}

Несколько рамок у одного блока при помощи множественных теней

.block {
    box-shadow: inset 0 0 0 1em #aaa,
                            inset 0 0 0 2em #f55,
                            inset 0 0 0 3em #eee;
}

Текстовые тени

Свойство text-shadow (поддержка IE10+)

.text {
    text-shadow: 0.3em 0.3em 0.2em #bbb;
}
Hello, World!

Синтаксис:

text-shadow: h-shadow v-shadow [blur] [color];
  • h-shadow - смещение по горизонтали ( > 0 - вправо, < 0 - влево)
  • v-shadow - смещение по вертикали ( > 0 - вниз, < 0 - вверх),
  • [blur] - размытие
  • [color] - цвет

Текстовые тени

Винтажный заголовок при помощи текстовой тени

.text {
    text-shadow: 4px 3px 0 #fff,
                            8px 6px 0 #ccc;
}
Hello, World!

Внутренняя тень у текста

У свойства text-shadow нет параметра inset. Один из способов сделать внутреннюю тень у текста:

.text {
    color: rgba(0, 0, 0, 0.4);
    text-shadow: 1px 4px 6px #fff,
                            0 0 0 #000,
                            1px 4px 6px #fff;
}
Hello, World!

Линейные градиенты

Градиент — это функция, показывающая направление наискорейшего возрастания некоторой величины, значение которой меняется от одной точки пространства к другой.

В CSS градиенты используются для задания плавного перехода от одного цвета к другому.

.block {
    background-image: linear-gradient(#e65100, #fff9c4);
}

Направление градиента по умолчанию: сверху вниз.

Поддержка браузерами: IE10+, Android 4.0, 4.1, 4.3 - с префиксом -webkit.

Направление градиента

Ключевое слово to с указанием направления: top, bottom, left, right.

.block {
    background-image: linear-gradient(
        to right,
        #e65100,
        #fff9c4
    );
}

Градиент по диагонали: достигается комбинированием слов top, bottom, left, right.

.block {
    background-image: linear-gradient(
        to left bottom,
        #e65100,
        #fff9c4
    );
}

Градиенты под углом

Направление в градусах задается ключевым словом deg.

.block {
    background-image: linear-gradient(
        270deg,
        #e65100,
        #fff9c4
    );
}

Углы градиента рассчитываются при помощи окружности:

Градиенты с несколькими цветами

В градиент можно включать несколько цветов:

.block {
    background-image: linear-gradient(
        to right,
        blue,
        red,
        yellow
    );
}

Пропорции цветов. Колорстопы

По умолчанию цвета в градиентах распределяются равномерно. Чтобы изменить это поведение, нужно использовать колорстопы.

Колорстопы записываются сразу после значений цветов. Их можно задавать в %, px и других единицах.

Поведение по умолчанию:

.block {
    background-image: linear-gradient(
        to right,
        #e65100,
        #fff9c4
    );
}

Колорстопы заданы явно:

.block {
    background-image: linear-gradient(
        to right,
        #e65100 150px,
        #fff9c4 75%
    );
}

Резкие переходы цветов

Если у соседних цветов задать одинаковые колорстопы, то получится резкий переход цветов:

.block {
    background-image: linear-gradient(
        white 50px,
		blue 50px,
		blue 100px,
		red 100px
    );
}

Повторяющийся линейный градиент

.block {
    background-image: repeating-linear-gradient(
        #ff5722 0,
        #ff5722 25px,
        #ffab91 25px,
        #ffab91 50px,
        #ffccbc 50px,
        #ffccbc 75px
    );
}

Радиальные градиенты

Радиальный градиент в CSS:

.block {
    background-image: radial-gradient(
        #fff,
        #e65100
    );
}

Явное задание радиуса градиента:

.block {
    background-image: radial-gradient(
        circle 5em,
        #fff,
        #e65100
    );
}

Позиционирование радиального градиента

Ключевое слово at с указанием единиц измерения, аналогичных свойству background-position:

.block {
    background-image: radial-gradient(
        circle 25em at 100% 0,
        #fff,
        #e65100
    );
}

Резкие переходы цветов

Тот же принцип, что и у линейных градиентов: одинаковые колорстопы у соседних цветов:

.block {
    border-radius: 50%;
    background-image: radial-gradient(
        circle,
        #ff5722 25px,
        #ffab91 25px,
        #ffab91 50px,
        #ffccbc 50px
    );
}
