Стажировка HTML5

Дмитрий Мещеряков

Цвет фона и фоновое изображение

Свойство background-color

body {
    background-color: #ff0000;
}

Форматы задания цветов в вебе:

  • шестнадцатеричный: #fa452e; #000;
  • RGB: rgb(0, 150, 0);
  • RGBA: rgba(0, 0, 0, 0.5);
  • цветовые константы: red, blue, green...

Свойство background-image

body {
    background-image: url('images/bg.jpg');
}

Повторение фонового изображения

Свойство background-repeat

.block {
    background-repeat: repeat; /* по умолчанию */
}
.block {
    background-repeat: no-repeat;
}
.block {
    background-repeat: repeat-x;
}
.block {
    background-repeat: repeat-y;
}

Позиционирование фонового изображения

Свойство background-position

.block {
    background-position: x y;
    /* x - расположение по горизонтали,
       y - по вертикали */
}

Допустимые значения x и y: px, %, em, rem, ключевые слова left, center, right, top, bottom.

.block {
    background-position: 0 0;
}
.block {
    background-position: 50% 50%;
}
.block {
    background-position: left bottom;
}

Фиксированное фоновое изображение

Свойство background-attachment

.block {
    background-attachment: scroll;
    /* фон прокручивается
       вместе с содержимым */
}
.block {
    background-attachment: fixed;
    /* фон не прокручивается,
       зафиксирован */
}

Сокращенная запись

background: [bc] [bi] [br] [bp] [ba];

  • [bc] - background-color
  • [bi] - background-image
  • [br] - background-repeat
  • [bp] - background-position
  • [ba] - background-attachment

Порядок свойств не важен.

Множественный фон

body {
    background-image: url('bg-image1.jpg'),
                                        url('bg-image2.jpg');
}

Выше будет то изображение, которое раньше в списке.

Размер фонового изображения

Свойство background-size

.block {
    background-size: x [y];
    /* x - ширина фонового изображения,
         y - высота */
}

Если высота не указывается, то она определяется автоматически в зависимости от ширины и исходных пропорций.

По умолчанию оба аргумента равны auto.

background-size: auto;

Исходные размеры изображения.

background-size: 100%;

Ширина изображения совпадает с шириной блока.

background-size: 100% 100%;

Изображение полностью вписывается в блок.

background-size: contain;

Изображение целиком вписывается в блок, часть блока может оказаться пустой.

background-size: cover;

Ширина или высота изображения равна ширине или высоте блока, часть изображения может обрезаться.

Границы фонового изображения

Свойство background-origin

Не применяется, когда задано свойство background-attachment: fixed;
.block {
    box-sizing: border-box;
    width: 16em;
    height: 16em;
    padding: 2em;
    border: 2em solid rgba(0, 0, 0, 0.5);
    font-size: 16px;
    background-size: auto;
    background-repeat: no-repeat;
}
.block {
    background-origin: padding-box;
}

Фон позиционируется относительно края элемента с учетом толщины рамки (значение по умолчанию).

Границы фонового изображения

.block {
    background-origin: border-box;
}

Фон позиционируется относительно границы, при этом линия границы может перекрывать изображение.

.block {
    background-origin: content-box;
}

Фон позиционируется относительно содержимого элемента (учитываются рамки и внутренние отступы).

Обрезка фонового изображения

Свойство background-clip

.block {
    background-clip: border-box;
}

Фон выводится под границами, совсем не обрезается (значение по умолчанию).

.block {
    background-clip: padding-box;
}

Фон обрезается по внутреннему краю области рамки.

Обрезка фонового изображения

Свойство background-clip

.block {
    background-clip: content-box;
}

Фон обрезается по краю области содержимого.
